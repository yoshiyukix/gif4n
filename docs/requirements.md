# 要件定義書

**プロジェクト名**: gif-to-note  
**バージョン**: 1.0  
**作成日**: 2026-03-29

---

## 1. プロジェクト概要

### 1.1 目的

AIツールが作成した動画（MP4 等）を、Note のテキスト記事に貼り付け可能な GIF ファイルへ変換するスマホアプリを開発する。

### 1.2 背景

Note のテキスト記事では GIF アニメーションを挿入できるが、動画から GIF を作成するには専用ツールが必要で、スマホから手軽に行える手段が限られている。本アプリはその操作をスマホで完結させる。

### 1.3 対象ユーザー

Note でテキスト記事を執筆するクリエイター・ブロガー

---

## 2. 外部制約条件

Note への GIF 貼り付けに関して、以下の制約が確認されている。

| 制約項目           | 内容                                  |
| ------------------ | ------------------------------------- |
| 対応記事種別       | テキスト記事のみ（画像記事は不可）    |
| 最大ファイルサイズ | **10 MB**（本アプリの上限として採用） |
| 横幅               | 620px 超は Note 側で自動縮小          |

---

## 3. 機能要件

### 3.1 動画入力

| ID    | 要件                                                 |
| ----- | ---------------------------------------------------- |
| F-001 | カメラロール（フォトライブラリ）から動画を選択できる |
| F-002 | ファイルアプリ等から動画ファイルを選択できる         |
| F-003 | 対応フォーマットは MP4 を必須とし、MOV も対応する    |

### 3.2 トリミング

| ID    | 要件                                                 |
| ----- | ---------------------------------------------------- |
| F-011 | 選択した動画を画面上でプレビュー再生できる           |
| F-012 | スライダーで開始・終了時間を指定してトリミングできる |
| F-013 | トリミング範囲のプレビューを確認できる               |

### 3.3 GIF 変換

| ID    | 要件                                                                                         |
| ----- | -------------------------------------------------------------------------------------------- |
| F-021 | フレームレートと解像度を自動調整し、10 MB 以内で最高品質の GIF を生成する                    |
| F-022 | 変換は以下の品質設定を優先順に試行し、最初に設定済みの最大サイズ以内に収まったものを採用する |
| F-023 | 変換処理はオンデバイスで完結し、ネットワーク接続を必要としない                               |
| F-024 | 変換の進捗をプログレスバーで表示する                                                         |
| F-025 | 変換中にキャンセル操作ができる                                                               |
| F-026 | 全品質設定でも最大サイズを超える場合はエラーを表示し、動画が長すぎる旨を案内する             |

**F-022 品質試行順位**

| 優先順位 | 解像度 | fps |
| -------- | ------ | --- |
| 1        | 620px  | 15  |
| 2        | 620px  | 10  |
| 3        | 480px  | 15  |
| 4        | 620px  | 5   |
| 5        | 480px  | 10  |
| 6        | 320px  | 15  |
| 7        | 480px  | 5   |
| 8        | 320px  | 10  |
| 9        | 320px  | 5   |

### 3.5 設定

| ID    | 要件                                                                        |
| ----- | --------------------------------------------------------------------------- |
| S-001 | GIF 変換の最大ファイルサイズを 5 MB / 7 MB / 10 MB から選択できる           |
| S-002 | 設定はアプリ再起動後も保持される                                            |
| S-003 | アプリバージョンを確認できる                                                |
| S-004 | プライバシーポリシー（オンデバイス処理・外部送信なし）を確認できる          |
| S-005 | 利用している OSS のライセンスを一覧で確認でき、各ライセンス全文を参照できる |

### 3.4 変換結果の利用

| ID    | 要件                                                            |
| ----- | --------------------------------------------------------------- |
| F-031 | 変換後の GIF をアプリ内でアニメーション再生して確認できる       |
| F-032 | 生成された GIF のファイルサイズと解像度を画面に表示する         |
| F-033 | GIF をカメラロールに保存できる                                  |
| F-034 | GIF をシステム共有メニューから他のアプリ（Note 等）へ共有できる |

---

## 4. 非機能要件

### 4.1 パフォーマンス

| ID     | 要件                                                           |
| ------ | -------------------------------------------------------------- |
| NF-001 | 30 秒動画（620px / 15fps）の変換を 30 秒以内に完了する         |
| NF-002 | 変換前にファイルサイズを事前推定し、余分な変換試行を最小化する |

### 4.2 互換性

| ID     | 要件                      |
| ------ | ------------------------- |
| NF-011 | iOS 15 以上で動作する     |
| NF-012 | Android 10 以上で動作する |

### 4.3 オフライン対応

| ID     | 要件                                         |
| ------ | -------------------------------------------- |
| NF-021 | すべての機能がネットワーク接続なしで動作する |

### 4.4 プライバシー

| ID     | 要件                                                   |
| ------ | ------------------------------------------------------ |
| NF-031 | 動画・GIF データを外部サーバーへ送信しない             |
| NF-032 | カメラロールへのアクセス権限を利用前に明示的に要求する |

| SCR-005 | 完了画面              | GIF プレビュー・サイズ表示・保存・共有                                  |
| SCR-006 | 設定画面              | 最大ファイルサイズ選択・バージョン表示・プライバシーポリシー確認        |

## 6. 技術要件

| 項目                    | 採用技術                                                 |
| ----------------------- | -------------------------------------------------------- |
| フレームワーク          | React Native (TypeScript)                                |
| プロジェクト管理        | Expo Dev Build（EAS）                                    |
| 変換エンジン（iOS）     | AVFoundation + ImageIO（ネイティブ、外部ライブラリ不要） |
| 変換エンジン（Android） | gifski（Rust 製、React Native ネイティブモジュール経由） |
| 変換方式                | Platform Channel 経由でネイティブモジュールを呼び出し    |
| ナビゲーション          | React Navigation                                         |
